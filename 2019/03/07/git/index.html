git命令

创建仓库

git init
添加远程仓库
git remote add 文件夹 URL
查询目录下所有项目
git remote -v 文件夹 URL
下载代码到本地
git clone URL
查看apk版本信息
aapt文件下 aapt dump badging apk名称
下载指定版本 进入文件目录后
git checkout ***

上传本地项目到华为云

git init //初始化
git remote add origin 仓库地址 //关联远程仓库
git pull origin master –allow-unrelated-histories //拉取库里数据
git add . //上传数据
git commit -m “first commit” //提交文件
git push -u origin master //推送文件

上传项目到coding

git init
git add .
git commit -m “xx”
git remote add origin https://git.coding.net/用户名/项目名.git
git push origin master
（git push –f origin master）
